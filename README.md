- how to use
	+ sty/time24.sty
		
		Mike Piffのtime.styを24h表記にしました｡`\nowtf`で24h表記になります｡`\fullnow`コマンドは`\the\year\ \the\month\slash\the\day\ \nowtf`を表示します｡これはpTeXのみなあらずLuaTeX､XeTeXでも使えます｡

	+ luapkgs.tex

		`\input{luapkgs}`

		`\editdate`で`\fullnow`と同様の表示が得られます｡
	
	+ xelpkgs.tex

		`\input{xelpkgs}`
